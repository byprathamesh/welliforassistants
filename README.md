## Getting Started

### Prerequisites
- Node.js & npm installed

### Installation
```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd home-health-connect

# Install dependencies
npm install

# Start the development server
npm run dev
```
